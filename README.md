# gotube-dl

### Run

* How to run the program

```
go run ./cmd -url=<video url> -key=<api key> -maxResult=<return a limited number of results>
```

- *url*  
Flag: -url  
Type: string  
Default: false  

- *key*  
Flag: -url  
Type: string  
Default: :)

- *maxResult*  
Flag: -maxResult  
Type: string  
Default: 1
